## BOM

- 2 PCB - necessary
- 2 Pico-like MCU board - necessary
- 2 USB-8 "JBL" TH connectors - necessary
- 42 Kailh or Gateron HS sockets - necessary
- 2 0208 resistors - optional
- 2 323 protection diodes - optional
